Getting Started

✶

Revenant runs as a local TCP proxy between your game client and the Simutronics servers. Point Wrayth, Avalon, Genie, or any SGE-compatible frontend at 127.0.0.1:4900 and Revenant handles the rest.

Lua scripts are loaded at startup and run as coroutines — no blocking, no lag. The monitor feature flag enables a native GUI launcher with saved accounts and character selection.

Requirements:

  • Rust 1.75+ — rustup.rs
  • A Simutronics account (GemStone IV or DragonRealms)
  • Windows 10/11, macOS, or Linux
Terminal — Invocation 
# Clone the repository
$ git clone https://github.com/Sordal-GSIV/revenant
$ cd revenant/engine

# Build release binary
$ cargo build --release

# Run headless
$ ./target/release/revenant \
    --listen    127.0.0.1:4900 \
    --account   YOURNAME \
    --password  YOURPASS \
    --game      GS3 \
    --character CHARNAME

# Or launch the GUI monitor
$ cargo run --features monitor
